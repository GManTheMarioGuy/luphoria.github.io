Skin creator with "GUI"

AKA real-time updating in the game.

To use

1: Drag this link to your Bookmarks bar: Add GUI HTML
2: Click on the bookmarklet when on sm64js.com. 3:Open the console and paste this JS code:
function update() {
i = 1;
while(i <= 6) {
x = document.getElementById("value" + i).value;
document.getElementById("rangeDisplay" + i).innerHTML = x;
i++;
}
skinType = document.getElementById("skinTypes").value;
window.myMario.skinData[skinType] = [document.getElementById('value1').value,document.getElementById('value2').value,document.getElementById('value3').value,document.getElementById('value4').value,document.getElementById('value5').value,document.getElementById('value6').value]
}

It should then work.
